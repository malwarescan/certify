# DEV DIRECTIVE — Remove Aggression, Unify Surface

## 1. Primary Button — Remove the "Crypto Glow"

**Fix:**
- Flatten shadow stack to: `0 6px 14px rgba(37,99,235,0.10), 0 0 0 1px rgba(255,255,255,0.35) inset`
- No external halo layer. No glow blur. No outer aura.
- Reduce gradient contrast 10–15%
- Hover: slightly darken, slightly increase shadow depth. Not glow.

## 2. Header / Nav — One Surface

- Remove header-specific gradient, blur, inset highlight
- Header becomes content on top of the same surface
- All plate background on .hero-plate only

## 3. Remove "UI Frosted App" Vibe

- No blur, no saturate > 1.05 on header
- Precision interface, not glass dashboard

## 4. Test After Change (1440px)

- Does the button look calmer than the headline?
- Does the header feel like part of the hero?
- Does nothing glow more than the emerald check?

## 5. Design Principle Moving Forward

**No glows. No halos. No neon. No stacked shadows.**

Hierarchy from: weight, contrast, spacing, discipline. Not light effects.
