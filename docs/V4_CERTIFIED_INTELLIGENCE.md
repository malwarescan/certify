# Certificate.now v4: The Certified Intelligence Platform

## Design Philosophy: "Clarity Meets Luxury"

Merges Microsoft Clarity's data intelligence with luxury authentication.

---

## Implemented

### Hero Section
- "From Doubt to Certainty with Blockchain" headline
- Certified gradient text effect (sapphire to emerald)
- Trust badges: Bank-grade NFC, Blockchain secured, AI-authenticated, 2M+ items
- Dual CTA: Start Verifying / See How It Works
- Holographic seal placeholder with NFC Tap Zone

### Trust Metrics (Clarity-style)
- 2M+ Items Certified
- $4.2B Total Value
- 99.9% Accuracy

### Certified Gradient System
- `.certified-gradient` - Background (sapphire to blue to emerald)
- `.certified-gradient-text` - Text gradient
- `.holographic` - Animated shimmer
- `.animate-nfc-pulse` - NFC tap animation

### Vault Dashboard
- Period selector (Last 30 days)
- 3 metric cards: Total Value, Items, Verified %
- Value breakdown with progress bars (Gems 68%, Watches 25%, Jewelry 6%)
- Recently verified with pulse indicator
- AI Insights section (insurance update, appraisal renewal)
- Quick actions, collections, trending

### MetricCard Component
- Reusable Clarity-style data card
- Value, label, optional change indicator
