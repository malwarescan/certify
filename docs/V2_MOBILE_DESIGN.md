# Certificate.now v2: Mobile-First Product Library

## Core Concept: "The Luxury Vault in Your Pocket"

Pinterest-style product library, immersive brand browsing, social verification feed.

---

## Navigation (Bottom Tab Bar)

| Tab | Route | Purpose |
|-----|-------|---------|
| 🏠 Vault | /vault | Personal collection, stats, recent verifications |
| 🔍 Search | /search | Discover, filter by category/brand/price |
| ⚡ Verify | /verify | NFC tap (hero action) |
| 💼 Library | /library | Browse all products, Pinboard grid |
| 👤 Me | /me | Profile, settings, security |

---

## Screens Implemented

1. **Vault** – Dashboard with $47.5K value, 12 items, quick actions, recently verified, collections
2. **Library** – Category pills, brand filters, masonry Pinboard grid
3. **Verify** – NFC tap zone (dark variant), recent verifications, alternative methods
4. **Search** – Filters (category, brand, price, verification), sort options
5. **Me** – Profile card, menu (certificates, insurance, notifications, security)
6. **Product Detail** – Full-screen image, verified badge, specs, provenance, blockchain
7. **Brand Page** – Hero, stats, collections, featured items
8. **Store Page** – Store info, in-store NFC zone, inventory

---

## Design System (Mobile Dark)

- **Background**: Sovereign Navy (#0F172A)
- **Cards**: Slate 800 (#1E293B) with 50% opacity
- **Accent**: Emerald (#10B981) for verified, CTAs
- **Text**: Slate 50 primary, Slate 400 secondary
